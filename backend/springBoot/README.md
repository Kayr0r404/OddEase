# OddEase Spring Boot Application

Placeholder Spring Boot 4 microservice within the OddEase platform.

## Prerequisites

- Java 17+
- Maven

## Tech Stack

- **Spring Boot 4.0.6** - application framework
- **Spring Data JPA** - PostgreSQL persistence
- **Lombok** - boilerplate reduction
- **PostgreSQL** - database (runtime dependency)

## Build & Run

```bash
cd backend/springBoot/OddEase
mvn clean install
mvn spring-boot:run
```
